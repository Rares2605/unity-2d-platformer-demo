# Unity 2D Platformer Demo

A simple 2D platformer prototype built in Unity and C#.

## Gameplay Demo

[(Media/screenshot.png)](https://youtu.be/mVeuOrKVq8M)

## Features

- Player movement left/right
- Flappy Bird-style jump
- Ground check system
- Camera follow
- Coin pickup system
- Score UI
- Health UI with 3 hearts
- Enemy patrol system
- Enemy damage cooldown
- Game Over panel
- Restart system with R key
- Freeze time on Game Over
- Coin pickup sound effect
- Take damage sound effect
- Finish trigger
- Level Complete panel with collected coins

## What I learned

- Unity 2D physics
- Rigidbody2D movement
- Trigger and collision detection
- UI updates
- Scene restart with SceneManager
- Time.timeScale for freeze time
- Basic enemy patrol logic
- Organizing a small Unity project

## Controls

- A / D or Arrow Keys: Move
- Space: Jump
- R: Restart after Game Over or Level Complete
